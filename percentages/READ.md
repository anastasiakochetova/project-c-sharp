# Калькулятор банковских процентов

В этой задаче реализовано консольное приложение для расчёта суммы вклада с ежемесячной капитализацией процентов.

## Описание

Программа принимает от пользователя три числа через пробел:
- исходную сумму вклада,
- годовую процентную ставку (в процентах),
- срок вклада в месяцах.

Рассчитывается итоговая сумма с учётом капитализации: в конце каждого месяца к текущей сумме прибавляется процент, рассчитанный как 1/12 от годовой ставки.

**Важно:** решение реализовано без использования циклов — расчёт выполняется по формуле сложных процентов.

## Формат ввода

Три числа через пробел:
```
<сумма> <процентная_ставка> <срок_в_месяцах>
```

### Пример ввода:
```
100.00 12 1
```

### Пример вывода:
```
101
```

> Пояснение: 12% годовых = 1% в месяц. За 1 месяц на 100 рублей начисляется 1 рубль → итого 101 рубль.

## Использование

1. Запустите консольное приложение.
2. Введите три числа через пробел.
3. Программа выведет накопленную сумму на момент окончания вклада.

## Реализация

Основная логика вычислений вынесена в метод `Calculate`, который:
- принимает строку с входными данными,
- парсит значения,
- применяет формулу сложных процентов:  
  `итог = начальная_сумма * (1 + годовая_ставка / 100 / 12) ^ количество_месяцев`,
- возвращает результат в виде целого числа (сумма в рублях без копеек).

> Примечание: согласно примеру, результат округляется **вниз до целого числа** (дробная часть отбрасывается).
