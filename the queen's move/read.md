# Шахматный валидатор хода ферзя ♛

Простая и элегантная программа для проверки корректности хода ферзя в шахматах.

## Описание

Программа помогает младшему брату Васи научиться правильно ходить ферзем, проверяя валидность хода согласно шахматным правилам.

## Правила хода ферзя

Ферзь может ходить:
- На любое количество клеток **по вертикали**
- На любое количество клеток **по горизонтали** 
- На любое количество клеток **по диагонали**

## Использование

```python
# Проверка хода
is_valid_queen_move("a1", "a8")  # True - вертикальный ход
is_valid_queen_move("d4", "h8")  # True - диагональный ход  
is_valid_queen_move("e5", "b5")  # True - горизонтальный ход
is_valid_queen_move("a1", "b3")  # False - ход коня
```
