# Склонение существительного "рублей"

Решение задачи по склонению существительного "рублей" в зависимости от числительного.

## Задание

Написать метод, склоняющий существительное «рублей» следующее за указанным числительным.

Например:
- для аргумента `10` метод должен вернуть `«рублей»`
- для `1` — вернуть `«рубль»`
- для `2` — `«рубля»`

## Решение

Метод учитывает следующие правила склонения:
- `рубль` — при числе, оканчивающемся на 1 (кроме 11)
- `рубля` — при числе, оканчивающемся на 2, 3, 4 (кроме 12, 13, 14)
- `рублей` — во всех остальных случаях
