# FieldsParserTask

**Цель:** Реализовать парсер для разделения строки на поля с поддержкой цитируемых значений и управляющих последовательностей.

**Задача:** Создать класс `FieldsParserTask` с методом `ParseLine`, который корректно обрабатывает:
- Обычные поля (разделенные пробелами)
- Поля в кавычках
- Экранирование кавычек внутри цитируемых полей (`""` → `"`)

**Название:** Парсер полей CSV/TSV-подобных строк

**Описание работы:** Метод последовательно анализирует входную строку, определяя тип каждого поля (обычное или цитируемое), извлекает их значения, пропускает разделители-пробелы и возвращает список полей.
