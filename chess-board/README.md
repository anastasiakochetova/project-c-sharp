# Шахматная доска произвольного размера

## Описание задачи
Программа генерирует шахматные доски нестандартного размера. Исходный код умел создавать только стандартную доску 8×8, но теперь был переработан для поддержки досок любого размера.

## Формат вывода
- Белые клетки обозначаются точкой (`.`)
- Черные клетки обозначаются шарпом (`#`)
- Каждая строка доски нумеруется
- Паттерн чередования клеток начинается с черной клетки в первой строке

## Пример вывода для доски 5×5:
```
1)#.#.#  
2).#.#.  
3)#.#.#  
4).#.#.  
5)#.#.#
```
## Реализация
Метод `WriteBoard(int size)` принимает размер доски и выводит ее в консоль согласно указанному формату.

## Использование
```csharp
public static void Main()
{
    WriteBoard(8);  // доска 8×8
    WriteBoard(1);  // доска 1×1
    WriteBoard(2);  // доска 2×2
    WriteBoard(3);  // доска 3×3
    WriteBoard(10); // доска 10×10
}
```
Особенности реализации
Первая клетка первой строки всегда черная (#)

Цвета клеток чередуются как в шахматном порядке

Каждая следующая строка начинается с цвета, противоположного началу предыдущей строки
