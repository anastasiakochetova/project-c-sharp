Remove Leading Spaces
Программа для удаления пробельных символов в начале строки с использованием цикла while.

Описание проблемы
Враги вставили в начало каждого полезного текста множество бесполезных пробельных символов! Вася смог справиться с ситуацией, когда такой пробел был один, но для обработки множественных пробелов требуется более надежное решение.

Решение
Программа использует цикл while для последовательной проверки каждого символа в начале строки и удаления всех пробельных символов до первого непробельного.

Ключевые особенности
Универсальность: Обрабатывает все виды пробельных символов:

Обычные пробелы

Табуляции

Переносы строк (\n, \r)

Эффективность: Использует минимальное количество итераций

Надежность: Корректно обрабатывает edge-cases (пустые строки, строки только из пробелов)
