# 🚀 Основы программирования | Юрий Юлернович

Репозиторий с заданиями и проектами из курса **"Основы программирования"** от Юрия Юлерновича.

## 📚 О курсе

Практико-ориентированный курс, посвященный фундаментальным основам программирования. Каждое задание — это шаг от простого к сложному, от базовых конструкций к полноценным приложениям.
## 🛠 Технологический стек

- **Язык**: C# / .NET
- **Парадигмы**: Императивное и ООП
- **Инструменты**: Visual Studio, Git
- **Концепции**: Алгоритмы, структуры данных, архитектура

## 💡 Особенности подхода

- 🔥 **Learning by doing** — минимум теории, максимум практики
- 🎯 **Постепенное усложнение** — от "Hello World" до физических симуляторов
- 🧩 **Модульность** — каждое задание самодостаточно
- ⚡ **Актуальные технологии** — современный C# и лучшие практики

## 🏆 Чему научитесь

- ✅ Фундаментальные концепции программирования
- ✅ Принципы ООП и проектирования архитектуры
- ✅ Работа с алгоритмами и структурами данных
- ✅ Навыки отладки и тестирования кода
- ✅ Основы физического моделирования

## 📖 Как использовать

1. Клонируйте репозиторий
2. Откройте решение в Visual Studio
3. Изучайте задания последовательно
4. Запускайте, модифицируйте, экспериментируйте!
